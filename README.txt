Conference Management System
A web application designed to manage the end-to-end lifecycle of scientific conferences.

•  Researcher Module
- Submissions: Handle initial and final camera-ready article versions.
- Notifications: Access submission guidelines and reviewing results.

•  Reviewer Module
- Selection: Browse and select articles for peer review.
- Conflict Management: Flag conflicts of interest and submit evaluation results.

•  Administrator Module
- User Management: Manage accounts and system access rights.
- Oversight: Manage assignments and final decision broadcasting.
